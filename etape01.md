# Etape 1

Étape 1 : intégrer les pages dans le projet Symfony.

Pour cela, le template à intégrer doit être récupéré depuis le dossier `files/`.

Vous devez donc créer les pages nécessaires afin d’intégrer le template.

Les pages à réaliser sont :

- Page de connexion : [files/login.html](files/login.html)
- Page de profile : [files/profile.html](files/profile.html)
- Details d'un produit : [files/product_details.html](files/product_details.html)
- Page accueil :  [files/index.html](files/index.html)
- Page des categories :  [files/browse_categories.html](files/browse_categories.html)
- Page de panier :  [files/cart.html](files/cart.html)
- Page des produits par categorie :  [files/products_by_category.html](files/products_by_category.html)

## Travail attendu : 

- Créer un projet Symfony.
- Intégrer les pages fournies dans le projet Symfony.
- Ne pas dynamiser les pages :
    - Pas de base de données requise.
    - Pas de code front requis (JavaScript).
