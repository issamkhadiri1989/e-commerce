# Etape 2

## Pré requis

Cette étape exige que [etape01.md](etape01.md).

Dans cette étape, vous avez intégré les pages dans un projet Symfony. 

Si vous n'avez pas encore fait le travail, il est temps de le faire. 

**Rappel :** Pour créer un projet Symfony il faut lancer les commandes suivantes :

```shell
composer create-project symfony/skeleton:8.0.* /chemain/vers/dossier

# // à lancer dans /chemain/vers/dossier
composer require webapp 
``` 

Puis réaliser le travail [etape01.md](etape01.md). 

## Travail attendu :

- Concentrez-vous sur les pages suivantes : `product_details.html`, `products_by_category.html` et `browse_categories.html`
